# Format TXTCR
Text Class Request

## Balise Format

- **`;[0-9]+#`** Format 1
- **`;|#`** Format 2

## Séparations

- **`;`** Valeurs
- **`:`** Key/Values in Dict

- **`[Sep]\`** Caractére d'échappement format 1
- **`\[Sep]`** Caractére d'échappement format 2

- **`;[0-9]+`** Balise basique format 1
- **`|;|`** Balise basique format 2

## Balises basique

- **`N#`** Name
- **`D#`** Description
- **`R#`** Affichage
- **`T#`** Time/Date
- **`B#`** Variable de Base
- **`E#`** Encodage
- **`H#`** Hash
- **`I#`** Info/Contenue

## Balises

- **`{`** Dict
- **`[`** List
- **`(`** Tuple
- **`"`** Str
- **`'`** Bytes
- **`=`** Calc
- **`+`** Int/Float Positif
- **`-`** Int/Float Negatif
- **`o`** None
- **`0`** False
- **`1`** True
- **`>`** Func
- **`#`** Class
